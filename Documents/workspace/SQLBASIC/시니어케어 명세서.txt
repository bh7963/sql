시니어 케어 프로젝트 명세서

### 로그인페이지

- 사용자는 아이디 비밀번호를 입력하여 일반 로그인을 할 수 있다.
- 아이디 비밀번호를 입력하여 로그인할 시 둘 중 하나라도 잘못 입력하였으면 '로그인 정보가 일치하지 않습니다.' 라는 문구를 출력한다.
- 사용자는 카카오, 네이버 로그인을 할 수 있다.

### 회원가입 페이지

- 사용자는 이름, 아이디, 비밀번호, 비밀번호 확인, 전화번호, 인증번호를 입력하여 일반 회원가입을 할 수 있다.
- 이름은 반드시 입력해야한다.
- 아이디는 중복확인을 해야한다. 존재하지 않는 아이디일 경우 '사용 가능한 아이디 입니다.'라는 문구를 출력하고, 존재하는 아이디일 경우 '이미 사용중인 아이디 입니다.'라는 문구를 출력한다.
- 비밀번호는 영문, 숫자를 혼용하여 8 ~ 13자를 입력해야하며 그렇지 않으면 '영문,숫자를 혼용하여 8 ~ 13자 입력해주세요.'라는 문구를 출력한다.
- 비밀번호 확인은 입력한 비밀번호와 일치해야한다. 일치하지 않으면 '비밀번호가 일치하지 않습니다'라는 문구를 출력한다.
- 전화번호는 숫자로만 11자리 입력해야한다. 숫자 11자리가 아니면 '숫자 11자 입력해주세요.'라는 문구를 출력한다.
- 전화번호로 인증번호를 전송에 성공하면 '인증번호가 전송되었습니다.'라는 문구를 출력한다.
- 인증번호 확인시에 인증번호가 일치하지 않으면 '인증번호가 일치하지 않습니다.'라는 문구를 출력하고 일치한다면 '인증번호가 확인되었습니다.'라는 문구를 출력한다.

### 고객 관리

### 고객 관리 리스트 페이지

전체 게시물 수, 현재 페이지와 전체 페이지가 표시된다.
고객 리스트는 고객번호, 고객명, 나이, 지역, 담당자, 수정 버튼, 삭제 버튼이 표시된다.
고객 리스트는 한 페이지에 최대 10개까지 표시되며 고객번호 기준으로 내림차순 정렬된다.
페이지 번호는 한번에 10개까지 표시된다.
제목을 통해 검색을 할 수 있다.

### 고객 관리 등록 페이지

- 고객 이름, 생년월일, 담당자, 주소를 입력할 수 있다.
- 담당자는 검색 버튼을 클릭하면 담당자의 이름을 통해 담당자를 검색할 수 있는 모달창이 뜬다.
- 검색 결과로 표시되는 담당자는 아이디, 이름, 전화번호가 출력된다.
- 목록 중 하나를 선택할 수 있다.

### 고객 관리 상세 페이지

- 고객 사진, 고객 이름, 생년 월일, 담당자 이름, 주소가 표시된다.
- 관리 기록 리스트에는 날짜, 내용, 사용용품, 개수가 표시된다.
- 현재 로그인한 사용자가 해당 고객의 담당자라면 기록을 작성할 수 있다.
- 기록엔 내용, 사용용품, 갯수를 입력한다.
- 현재 로그인한 사용자가 해당 고객의 담당자라면 수정 버튼을 클릭하여 고객 수정 페이지로 이동할 수 있다.
- 현재 로그인한 사용자가 해당 고객의 담당자라면 삭제 버튼을 클릭하여 고객을 삭제할 수 있다.

### 고객 관리 수정 페이지

- 수정 페이지에 접근하면 기존의 고객 사진, 고객이름, 생년월일, 담당자, 주소가 입력되어 있다.
- 고객 이름, 생년월일, 담당자, 주소를 수정할 수 있다.
- 담당자는 검색 버튼을 클릭하면 담당자의 이름을 통해 담당자를 검색할 수 있는 모달창이 뜬다.
- 검색 결과로 표시되는 담당자는 아이디, 이름, 전화번호가 출력된다.
- 목록 중 하나를 선택할 수 있다.

### 용품 관리

- 전체 게시물 수, 현재 페이지와 전체 페이지가 표시된다.
- 용품 리스트는 용품번호, 영품명, 용도, 개수, 수정 버튼, 삭제 버튼이 표시된다.
- 용품 리스트는 한 페이지에 최대 10개까지 표시되며 용품번호 기준으로 내림차순 정렬된다.
- 페이지 번호는 한번에 10개까지 표시된다.
- 용품명을 통해 검색을 할 수 있다.
- 등록 버튼을 클릭하면 리스트 상단에 등록 컴포넌트가 나타난다.
- 용품 등록은 용품명, 용도, 개수를 입력할 수 있다.
- 용품 리스트에 표시된 각 항목에 수정 버튼을 클릭하면 리스트 상단에 수정 컴포넌트가 나타난다.
- 용품 수정은 용품명, 용도, 개수를 수정할 수 있다.
- 용품 리스트에 표시된 각 항목에 삭제 버튼을 클릭하면 용품을 삭제할 수 있다.

### 인사 관리

### 인사 관리 리스트 페이지

- 전체 게시물 수, 현재 페이지와 전체 페이지가 표시된다.
- 인사 리스트는 요양사 아이디, 요양사 이름, 전화번호, 수정 버튼이 표시된다.
- 각 항목에 수정버튼은 로그인한 사용자의 항목에 대해서만 수정 버튼이 표시된다.
- 페이지 번호는 한번에 10개까지 표시된다.
- 요양사 이름을 통해 검색을 할 수 있다.

### 인사 관리 상세 페이지

- 요양사의 아이디, 이름, 전화번호가 표시된다.
- 관리 중인 고객이 존재한다면 해당 고객들의 리스트가 표시된다.
- 관리 중인 고객 리스트는 고객번호 고객명 나이 지역이 표시된다.

### 인사 관리 수정 페이지

- 수정 페이지에 접근하면 기존의 아이디, 이름, 전화번호가 표시되며 이름은 표시되어 있다.
- 이름을 수정할 수 있다.
